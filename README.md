# 晚菘
![cabbage_project](res/cabbage.png)

研究字形结构的组成与分解方式，实现对西夏文、辽金大小字以及女书等的不同编码形式。

第一计划主要是研究基于五笔方式的西夏文输入法编码实现（基于Rime平台的码表制作），后期希望结合韵音实现混合码表的实现。

- [蕃字五笔（稿）](docs/KvinTangut.md)
